MATALB Coder

In this post we will discuss the use of MATLAB Coder to generate C/C++ code form existing .m files.

Often developers prefer high level interpretted languages like MATLAB, Python, etc. to prototype computational alogrithms. These implementations lack the performance to study large computational problems. Often, these prtotypes are reimplemented using compiled languages like C, C++ or Fortan to achieve performance. It requires substantial amount of time and effort for reimplementation and to verify the reimplementation.

MATLAB Coder provides a solution to the time consuming reimplementation problem. It provides tools to generate C/ C++ code for already working MATLAB source code. In this work we convert a simple Finite element time integrator project from MATLAB to C++, we will use MATLAB Coder to generate C++ code. The C++ code will be compiled using gcc. At the end, we will also compare the performance of both our MATLAB and C++ implementations.
